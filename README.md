# kpi-performance-analysis
End-to-end data analytics project analyzing business KPIs using SQL, Excel, and Power BI with interactive dashboards and actionable insights.
---

## 📂 Project Structure
- dataset/ → Raw data files
- sql/ → SQL queries for analysis
- powerbi/ → Dashboard file (.pbix)
- screenshots/ → Dashboard images

---

## 🛠 Tools Used
- SQL
- Excel
- Power BI
- ---

## 🧾 SQL Analysis

Key analysis performed using SQL:

- Calculated RPH (Revenue Per Hour)
- Compared actual vs target performance
- Categorized agents using CASE statements
- Ranked agents using window functions
